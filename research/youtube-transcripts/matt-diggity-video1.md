# Matt Diggity – AI SEO Topical Map Strategy (Transcript Notes)

In this session, Matt Diggity and Julian demonstrate how to build a keyword planning and topical mapping tool using AI.

They start by exporting keyword ideas from Google Keyword Planner and then uploading the CSV file into Claude AI to generate a topical map automatically.

Key ideas explained:

- Use Google Keyword Planner to collect keyword ideas.
- Export keyword data as a CSV file.
- Upload the file into Claude AI.
- Ask Claude to generate a topical map structure.

Important lesson from the video:

AI-generated topical maps are helpful but should NOT be followed blindly.

Instead:

- review keyword overlap
- remove duplicate intent keywords
- avoid keyword cannibalization
- merge similar topics into single pages when needed

Example:

“SEO agency” and “SEO company”
should often be ONE page instead of two separate pages.

Best practice explained:

Create hierarchy like this:

Parent topic → main service page  
Child topic → supporting service page  
Sub-child topic → deeper niche articles

Example structure:

Industry SEO Services
→ Law firm SEO
→ Dental SEO
→ SaaS SEO

Each child topic becomes its own page under the main category.

Final workflow shown in the video:

1. Export keywords from Google Keyword Planner
2. Upload CSV into Claude
3. Generate topical map
4. Refine keyword clusters
5. Remove overlaps
6. Build parent-child page structure
7. Use AI to speed up planning

Conclusion:

AI tools like Claude can replace expensive SEO tools for keyword clustering and topical map generation if used correctly.
